# Superstore Sales Dashboard – Power BI

## Project Overview

This project presents an interactive sales dashboard built using Power BI to analyze the Superstore dataset. The dashboard provides insights into sales performance, profit trends, and customer purchasing behavior across different regions and product categories.

The goal of this project is to demonstrate how business intelligence tools can be used to transform raw sales data into actionable insights.

---

## Dataset

The dataset used in this project is the **Superstore Sales dataset**, which contains information about:

* Orders
* Customers
* Product categories
* Sales
* Profit
* Regions and segments
* Order dates

---

## Dashboard Features

The dashboard provides insights into:

### Sales Performance

* Total Sales
* Total Profit
* Sales by Category
* Sales by Sub-Category

### Regional Analysis

* Sales by Region
* Profit distribution across regions

### Product Analysis

* Top performing product categories
* Sub-category sales comparison

### Time-Based Analysis

* Monthly sales trends
* Order trends over time
